# 质量检查清单

- [ ] 已按对话 60%、教材 40% 融合
- [ ] 对话决定讲解重点
- [ ] 教材完成补漏和校验
- [ ] 未把对话仅放在附录
- [ ] 未把教材写成机械摘要
- [ ] 每章一个文件夹
- [ ] 每个最小小节一个 Markdown
- [ ] 每个笔记记录 conversation_weight 与 textbook_weight
- [ ] Markdown 结构清晰，合理使用标题、列表、表格、callout、链接、代码块和检查清单
- [ ] 数学公式、递推式、复杂度、概率、矩阵或距离度量已用 LaTeX 表达
- [ ] 公式后解释了符号含义、适用条件和常见错误
- [ ] 核心知识点讲解详尽、有条理、实用，但没有无意义扩写
- [ ] 反复困惑触发深度升级
- [ ] 教材未讨论但对话重要的问题得到保留
- [ ] 对话未讨论但教材关键的内容得到补充
- [ ] 错误对话结论已纠正
- [ ] 已按当前主题性质补充带注释的高价值外部资源
- [ ] 外部资源类型合适，例如练习题、案例、数据集、工具、可视化、官方文档、论文、标准或权威参考
- [ ] 外部资源不是无注释链接列表
- [ ] 未复制长篇聊天原文
- [ ] 学习薄弱点已生成索引
- [ ] 数据集与外部资源已生成索引
